Rules
Do my bidding...
  • You will be given as a target a random number at the start of the game that you must match.
  • There are four crystals below. Each crystal is randomly assigned a fixed number for the duration of the game. By clicking on a crystal you will add that specific amount of points to your total score.
  • You win the game by matching your total score to the targeted random number set at game start, you lose the game if your total score goes above the random number.
  • The value of each crystal is hidden from you until you click on it. Remember each crystals number!
  • Each time when the game starts, the game will change the values of each crystal and the target
